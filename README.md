# ShelterOps

- HTML/CSS研修向けに制作した避難所簡易運営シミュレーション。
- [CirKit](https://cirkit.jp)での研修をもとに、HTML/CSSの技術を駆使して実装。

## 特徴

- HTML/CSSのみ
- JavaScript不使用
- ゲーミフィケーション要素
- Checkbox Hack使用
- レスポンシブ対応

## 使用技術

- HTML5
- CSS3
- Flexbox
- Grid Layout
- Animation
- Responsive Design

## テーマ

避難所運営を題材に、状態変化やUI設計を実装。

## 制作目的

- HTML/CSSのみで動的表現を追求。
- 研修教材としての活用。
